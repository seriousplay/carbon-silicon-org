# 碳硅组织设计工作室品牌架构

## 品牌定义

- 母品牌：碳硅组织
- 总入口：碳硅组织设计工作室
- 英文辅助：Carbon Silicon Org Studio
- 一句话价值：为 AI 时代领导者设计可自进化的碳硅共生组织。
- 支撑表达：从组织诊断，到蓝图生成，到业务回路落地。

## 产品层级

- 碳硅组织设计工作室：统一入口，承接 CEO 的组织进化工作流。
- 组织进化测评：采集组织画像、业务压力、AI 采用阶段和课程期待。
- 组织进化蓝图：生成组织聚焦方向、阶段判断和候选业务回路。
- 业务回路设计器：把选定回路转成可执行、可治理、可验证的人机协同方案。

## 域名空间

- 主入口：csi-org.com/loop-designer
- 兼容入口：loop.csi-org.com/loop-designer
- 品牌短入口预留：studio.csi-org.com
- 后续扩展：
  - academy.csi-org.com：课程与闭门会
  - docs.csi-org.com：方法论与交付文档
  - admin.csi-org.com：内部、讲师和运营后台

## 标识方向

总品牌标识采用“碳硅双环 + 组织蓝图网格”：

- 碳：稳定结构、组织骨架、治理边界。
- 硅：智能体、反馈回路、计算能力。
- 组织：节点和连线表示业务回路、角色关系和组织设计。

当前应用仍保留旧路径和旧工具能力，但可见品牌以“碳硅组织设计工作室”为主。
