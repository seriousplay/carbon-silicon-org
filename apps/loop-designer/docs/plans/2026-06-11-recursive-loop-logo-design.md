# Loop Designer Recursive Loop Logo

## Intent

The mark represents a business loop that does not merely repeat. Each completed
cycle feeds learning into the next cycle, producing a more capable version of
the operating system.

## Visual System

- A single continuous spiral-loop path communicates operation, feedback, and
  recursion without relying on small nodes or labels.
- The path moves from cyan to acid green: sensing and data become an improved
  business decision.
- A cyan origin point and an acid-green terminal arrow make the direction of
  evolution explicit.
- The open center preserves legibility at favicon size and reads as the next
  cycle waiting to begin.
- Rounded geometry matches the product's technical but human-centered tone.

## Deliverables

- Transparent icon mark for use inside the product UI.
- Dark rounded-square application icon for browser and platform surfaces.
- Reusable React component so all in-product placements stay consistent.

## Acceptance Criteria

- Recognizable at 16 px and 40 px.
- Uses only the existing ink, cyan, and acid-green product colors.
- Contains no text, fine node diagrams, shadows, or decorative particles.
- Appears in the home header, design workspace header, and site metadata.
